# 📚 BookScan 26.04.18

책 내용을 스캔하고, 정리하고, 다시 찾아보는 것을 돕는 웹 애플리케이션입니다.

텍스트 기반으로 책 내용을 저장하고, 검색·하이라이트·정리를 하나의 흐름으로 연결합니다.

---

## 🔧 주요 기능

### 1. 책 & 페이지 관리

* 책 생성 및 수정 / 삭제
* 페이지 단위로 텍스트 저장
* 페이지 순서 변경 기능

### 2. 검색

* 전체 책 내용 검색
* 책 내부 검색 (페이지 단위)
* 최근 검색어 자동 저장

### 3. 하이라이트

* 텍스트 선택 후 색상 하이라이트
* 색상별 필터링
* 하이라이트 위치로 바로 이동

### 4. 구조 기반 데이터 관리

* `localStorage` 기반 데이터 저장
* books / pages / highlights / tags / folders 구조 분리 

---

## 🧱 기술 스택

* HTML / CSS / Vanilla JavaScript
* LocalStorage (DB 대체)
* Node.js (정적 서버)

## 🚀 실행 방법

### 1. 설치

```bash
npm install
```

### 2. 실행

```bash
npm start
```

서버 실행 후:

```
http://localhost:3000
```

에서 확인 가능합니다. 

---

## 💾 데이터 구조

기본 데이터는 `data.js`에 정의되어 있으며, 초기 실행 시 localStorage에 저장됩니다. 

주요 컬렉션:

* books
* pages
* highlights
* tags
* folders
* recentSearches

---

## 🧠 검색 엔진

간단한 텍스트 기반 검색 로직을 사용합니다.

* 페이지 텍스트 기준 검색
* 위치, 빈도, 제목/저자 포함 여부 기반 점수 계산 

---

## ⚠️ 참고 사항

* 데이터는 브라우저 localStorage에 저장되므로
  브라우저를 변경하거나 캐시를 삭제하면 데이터가 사라질 수 있습니다.
* 서버는 정적 파일 서빙용으로만 사용됩니다.
