# Claude 작업 지침

공통 기준은 `AGENTS.md`를 따릅니다.
UI 세부값이 필요할 때만 `docs/ui-rules.md`를 참고합니다.

문서 내용이 충돌하면 `AGENTS.md`를 우선합니다.

## 핵심

- Codex 또는 다른 AI의 기존 작업을 먼저 읽고 이어서 작업합니다.
- 기존 화면, 기능, 반응형을 깨지 않습니다.
- 요청과 직접 관련 없는 파일은 수정하지 않습니다.
- 파일명, 클래스명, 함수명, 데이터 구조를 임의로 변경하지 않습니다.
- 긴 설명보다 짧은 결과 기록을 남깁니다.

## 작업 후

`수정 / 유지 / 확인 / 참고` 형식으로 짧게 공유합니다.
